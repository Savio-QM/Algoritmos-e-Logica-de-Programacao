# Algoritmos-e-Logica-de-Programacao
Postagens sobre o curso de algoritmos e lógica de programação que estou fazendo.

![image](https://user-images.githubusercontent.com/90512344/156780324-ea54d460-3605-4c73-bedd-03001cdbd7c4.png)
O VisualG é um programa que foi desenvolvido para que programadores iniciantes que falem portugês possam utilizá-lo para aprender o básico de programação. A lógica por trás da criação desse programa é que usar linguagens direto em inglês pode dificultar o entendimento da pessoa que está começando no mundo da programação, daí a solução para isso foi criar esse programa que utiliza a língua Portugol, o que significa que é basicamente uma linguagem de programação em português.
